Package: cmmetrics
Title: a package with functions for confusion matrix metrics, for evaluation of classificators performance in binomial and multinomial outcomes.
Version: 0.0.0.9000
Authors@R: person("Rafael", "B. Salviato", email = "rbsalviato@gmail.com", role = c("aut", "cre"))
Description: this package provides some functions that aims to analyze any type of classificator (e.g: logistic regressions, multinomial regressions, SVM, etc) performance through confusion matrices. The functions often describe some formulas, which symbols used are meant as it follows: TP and TN stands for "true positives" and "true negatives" respectively, and FP and FN stands for "false positives" and "false negatives" respectively.
